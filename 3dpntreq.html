Dual Color +1.50

Looks cooler, more expensive, Have to stay at the machine for manual change. Has chance of Delamination (splitting apart) and needs to be re-printed from scratch.

Because 3D printing is down from the bottom up only vertical layers can be changed. Requires a Pause command to be inserted in to the machine code to tell the machine when to stop. Manual color change is done and then the printing resumed.

Specialty Filament/Colors +$Varies

Specialty Filament like Wood PLA (sawdust mixed in with the PLA) has special considerations (Wood PLA clogs alot and can ruin hours of printing).

Wood PLA can be stained, sanded and painted.

Some like PLA+ are a little stronger but require higher temps and is more prone to clogging.

PLA+ in the color Nature/Clear/Lighter colors can sometimes burn and leave brown blobs

Infill Pattern and Density & Shells/Walls $ Varies

Infill Density determines how thick it is MOST of the name tags are done at 5% or 10%. This changes the weight, amount of material used, and time. It can increase strength or better appearing layers (some 5% can cause the upper layer to be seen to not fill in completely leaving a small hole in the surface).

Infill Pattern is how the infill is printed. This can have some very nice effects on see through items. Some patterns require special programs. Most of my name tags are done with Gyroid infill 5-25% depending on the name and how the software displays it to me, then I have to think about it and adjust acoordingly.

Shells/Walls are the real strength of the print. I use default 2 walls @ 0.8mm thick with 4 top & bottom @ 0.8mm. This can be increased for strength or to be water tight. This has no effect on things like the Containing Ring as it is only 1mm thick and is a top layer (Containing Ring is also 100% infill). The above photo was a print I stopped (the orange part) I was only testing different software and infills that software (not my main software) offers. You can see the infill and the shells/walls.

Supports (Standard required no cost) Other supports non-standard $ Varies and needed.

Support are some times required as you can't print just in the air. Supports use extra material and adds time. Standard supports included.

Supports not only use extra material and more time during printing the need Post-Processing (Manual Clean up and the tools to do it).

On other prints cleaning up the supports can break the entire part and requires re-printing.

Support also leave a rough surface behind.

In the orange one (upside down) supports are used in the two circle cut outs and the two rectangle cut outs. This keeps everything on the same level as 3D printing is bottom to top.

Another common support I have to use sometimes due to material like PLA+ or re-calibrating or re-adjust the machine is raft, the machine lays down many layers underneath before actually printing the part, Adds a lot of time and material and can leave a less than desirable finsh on the bottom of the print.

Direct Glass printing +$5.00
(Currently not available)

Printing directly on glass can give a mirror like finish on a lot of materials. However requires a lot of extra things, wide brim supports (extra time and supply) and tape to help adhere it or it will come loose and make a mess.

Only the bottom surface touching the glass benefits.

I'm using a PEI sheet now which helps elimintate the need for tape or glue stick not always the PLA+ is hit and miss.

I would need a second round glass bed that would be need to switch in and out with the PEI howerver this weakens the silicon mat the glass beds are placed on and then those would become a consumable/supply.

Speed and Quality and layer height
(Currently not available)

When you increase the speed quality goes down (bottom item in photo 300mm/sec 0.2mm layer height)
Slowed down quality gets better but takes longer (upper item in photo just at my standard 50mm/sec).

Layer Height usually I run at 0.2mm layer height 5 layers for 1mm in height. Decreaseing layer height increases time.

A Standard Name tag with minimal required supports at 0.2mm layer height print time is 1hr21 min no ironing.

The same Name tag at 0.1mm layer height print time is 2hr42 min.

The same Name tag at 0.05mm (50 Microns) layer height print time is 5hr25 min.

All with a 0.4mm nozzle. I could go down to a 0.1mm nozzle if I wanted the print to take two days.
